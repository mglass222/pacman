# Pac-Man

A browser-based Pac-Man game built with HTML, CSS, and JavaScript.

**[Play the game →](https://mglass222.github.io/pacman/)**

## Controls

- **Arrow keys** or **WASD** to move
- Click **PLAY** on the title screen to start

## Local development

Open `index.html` in a browser, or serve the folder with any static file server.
